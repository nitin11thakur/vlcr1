IVR Call Flow

Exotel IVR integration · Zero English literacy required · 12 language prompts

📞 IVR Step-by-Step Flow
1
Welcome & Language Prompt
Call answered. Welcome message played in Hindi. Citizen asked to speak language name or press 1–9.
Fallback: default Hindi after 5s silence
2
Language Detection
IndicLID identifies language from spoken response OR DTMF key maps to BCP-47 code. Language confirmed in citizen's language.
IndicLID ~200ms · min conf. 0.80
3
Complaint Recording
Prompt citizen to describe complaint after beep. Max 90 seconds. Extended by 30s if still speaking.
Max 120s · WAV 16kHz mono
4
Processing (Async)
Hold music plays. Bhashini ASR transcribes → IndicTrans2 translates → Claude API classifies → Routes to dept.
Target: p95 <60s end-to-end
5
Reference Number (TTS)
Reference number read aloud via Bhashini TTS in citizen's language. Fallback: Google Cloud TTS India voices.
TTS latency <2s
6
Additional Details / End
Citizen can press 1 to add more details or 2 to end. Auto-end after 10s silence.
Optional extension
7
SMS Acknowledgement
SMS sent to caller's phone with reference number in detected language. If SMS fails: number read again.
Twilio / Gupshup
